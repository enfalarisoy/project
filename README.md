# Paper Companion

Interactive paper companion for "Oops! Out of Moves".

## Files

- `index.html`: browser preview entry point
- `paper-companion-simple.jsx`: editable React/JSX source
- image assets used by the companion

## Run locally

From this folder:

```bash
python3 -m http.server 8123
```

Then open:

```text
http://127.0.0.1:8123/index.html
```
